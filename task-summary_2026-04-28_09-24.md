# 产品报价汇总表生成

## 任务
将3张产品报价图片（基础油/纯露/精油）合并为一个 Excel 文件保存到桌面。

## 执行过程
1. 读取图片提取产品数据
2. openpyxl 安装遇到 PEP 668 限制（--user 也被限制）
3. 使用 `pip3 install --break-system-packages openpyxl` 安装成功
4. 生成带格式化的 Excel 文件

## 输出
- 文件：`/Users/EricMac/Desktop/产品报价汇总表.xlsx`
- 共计 **110 个产品**：
  - 基础油/植物油：31种
  - 纯露/花水：20种
  - 精油：59种

## 表格格式
- 表头：蓝色背景+白色粗体字，冻结首行
- 分类行：淡蓝色背景，分组显示
- 边框+对齐：数值列居中、文字列左对齐
- 列宽：自动适配内容
