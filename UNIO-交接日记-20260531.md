## 九、特殊功能与已知问题

### 元香祭司（AI Oracle）

- 前端在 `src/site/pages/SiteOracle.tsx`
- API调用路径：`/api/oracle-proxy`（Vercel Serverless）
- Serverless转发到：`https://oracle.unioaroma.com/api/oracle`（阿里云）
- 阿里云上运行 Express 服务器（端口3456），调用 DeepSeek Chat API
- Prompt中定义了严格的角色「Unio元香祭司」、三段式回复格式、订购链接标记 `[订购产品名|code:XXX]`
- 复方问诊信息通过 `BlendingQuestionnaire.tsx` 表单提交

### 用户评价（Reviews）

- **当前状况**：158条评价写入 `src/site/data/reviews.ts` 静态文件
- **后台审批**：用 Supabase reviews 表查 `is_approved` → 如果为空则回退使用静态158条
- **尚未完成**：前后台评价完全打通（计划中）

### 字体优化（2026-05-07）

- `fonts.css` 子集化（`chinese-simplified+latin`）
- 构建时自动删除 .woff 旧格式
- 效果：dist 从 66MB → 29MB（-56%），字体文件从 1176个 → 10个

### 图片优化铁律

- 一次只改一个CSS属性（aspect-ratio / object-fit / scale / 背景色 / 遮罩）
- 改完确认再改下一个
- 缩略图默认从原始保守方案开始：`aspect-[3/4]` + `object-cover` + 纯白底 + 无scale + 无遮罩
- 如果用户说「恢复成之前」→ 找git记录拷贝原始版

### 对比度注意

- 用户说「太淡了」= 对比度不够 / 颜色太浅
- 字体无需变大，要提升文字色值
- 正文（14px+）提到 70-80% 黑
- 标签/辅助字提到 50-65% 黑
- 每次提升一次性拉 15-25%（不要 5% 小步调）
